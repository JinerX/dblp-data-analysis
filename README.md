# Purpose
A data analysis project for dblp  dataset found at: https://dblp.org/.

Download the xml file and extract it to the data folder for the code to work.

# Current stage
Currently finished disambiguating the different name variants and started working on the final database which will be used for actual data analysis.

# Setup
`docker compose up --build`

at `localhost:8888` you're going to have access to the jupyter notebook.
